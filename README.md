# TypeHero

Typing game.

- tracks WPM / accuracy
- uses random list of 1,000 words to generate each line
- keeps typing line centered on 2nd once reached
- mobile friendly

### What's Next?

- add user options
- more words / generation flavors
- colorschemes?
- improve virtual keyboard accuracy
- logo

### What this won't have

- leaderboards
- sign-in
- ads
